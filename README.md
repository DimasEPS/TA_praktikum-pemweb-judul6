# Weather Dashboard

Single-page dashboard untuk cuaca saat ini dan prakiraan 5 hari dengan pencarian kota, favorit, toggle 째C/째F, dan tema light/dark

## API

- Data cuaca & geocoding: [Open-Meteo](https://open-meteo.com)

## Fitur Singkat

- Cuaca saat ini (suhu, kelembapan, angin, feels-like, ikon kode cuaca).
- Prakiraan 5 hari (min/max + ikon).
- Cari kota dengan auto-suggest, simpan/hapus favorit (localStorage), refresh manual + auto 5 menit.
- Toggle 째C/째F dan tema gelap/terang (persist).
